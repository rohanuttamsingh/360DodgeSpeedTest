# 360Dodge SpeedTest

## Requirements

* Python 3.8.10
* TensorFlow 2.6.0
* NumPy 1.21.4
* H5py 3.1.0

## Instructions

1. Unzip `checkpoints.tar.gz` in `checkpoints` directory
1. Unzip `val.tar.gz` in `data` directory
1. Run `./speed_test.py -checkpoint_path checkpoints/latest -data_path data/val`
